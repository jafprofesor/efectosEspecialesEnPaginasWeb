Ejemplo Básico de Rollover

Este es un ejemplo básico de efecto rollover (cambio de imagen al pasar el cursor) utilizando JavaScript. Pasa el cursor sobre la imagen para ver el efecto.

Cómo funciona

El efecto se logra mediante eventos de JavaScript que cambian el atributo src de la imagen cuando el cursor pasa sobre ella: 

// HTML
<img id="imagenRollover" src="imagenes/boton-normal.svg" alt="Botón">

// JavaScript
const imagen = document.getElementById('imagenRollover');

imagen.addEventListener('mouseover', function() {
    this.src = 'imagenes/boton-hover.svg';
});

imagen.addEventListener('mouseout', function() {
    this.src = 'imagenes/boton-normal.svg';
});

Ventajas y desventajas

Ventajas:

  • Fácil de implementar
  • Compatible con navegadores antiguos
  • No requiere CSS avanzado

Desventajas:

  • La imagen secundaria se carga solo cuando se necesita, lo que puede causar un retraso
  • Requiere JavaScript para funcionar
  • Más código que la solución con CSS
