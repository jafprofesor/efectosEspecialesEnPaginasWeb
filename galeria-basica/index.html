Galería de Imágenes - Ejemplo Básico

Este es un ejemplo básico de una galería de imágenes implementada con JavaScript simple. Permite navegar entre varias imágenes utilizando los botones anterior y siguiente.

Imagen 1: Paisaje montañoso

Explicación del Código

HTML

<!-- Estructura básica de la galería -->
<div class="galeria">
  <button id="btn-anterior" class="btn-navegacion">❮ Anterior</button>
  
  <div class="contenedor-imagen">
    <img id="imagen-actual" src="imagenes/imagen1.jpg" alt="Imagen 1">
    <p id="descripcion-imagen" class="descripcion-imagen">Descripción de la imagen</p>
  </div>
  
  <button id="btn-siguiente" class="btn-navegacion">Siguiente ❯</button>
</div>

JavaScript

// Array con la información de las imágenes
const imagenes = [
  { src: 'imagenes/imagen1.jpg', alt: 'Imagen 1', descripcion: 'Descripción 1' },
  { src: 'imagenes/imagen2.jpg', alt: 'Imagen 2', descripcion: 'Descripción 2' },
  // Más imágenes...
];

// Variables para los elementos del DOM
let indiceActual = 0;
const imagenActual = document.getElementById('imagen-actual');
const descripcionImagen = document.getElementById('descripcion-imagen');

// Función para mostrar la imagen actual
function mostrarImagen(indice) {
  imagenActual.src = imagenes[indice].src;
  imagenActual.alt = imagenes[indice].alt;
  descripcionImagen.textContent = imagenes[indice].descripcion;
}

// Eventos para los botones de navegación
btnAnterior.addEventListener('click', function() {
  indiceActual = (indiceActual > 0) ? indiceActual - 1 : imagenes.length - 1;
  mostrarImagen(indiceActual);
});

btnSiguiente.addEventListener('click', function() {
  indiceActual = (indiceActual < imagenes.length - 1) ? indiceActual + 1 : 0;
  mostrarImagen(indiceActual);
});
