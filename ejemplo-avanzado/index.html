Ejemplo Avanzado de Rollover

Este es un ejemplo avanzado de efecto rollover utilizando CSS. Pasa el cursor sobre los botones para ver los diferentes efectos.

Rollover con CSS usando dos imágenes

Rollover con CSS usando una imagen sprite

Cómo funciona

El efecto se logra mediante CSS, sin necesidad de JavaScript:

/* HTML */
<div class="boton-rollover"></div>

/* CSS - Usando dos imágenes */
.boton-rollover {
    display: inline-block;
    width: 200px;
    height: 80px;
    background-image: url('imagenes/boton-normal.svg');
    background-size: cover;
    cursor: pointer;
}
.boton-rollover:hover {
    background-image: url('imagenes/boton-hover.svg');
}

También se puede implementar usando una sola imagen (sprite) que contiene ambos estados: 

/* HTML */
<div class="boton-sprite"></div>

/* CSS - Usando una imagen sprite */
.boton-sprite {
    display: inline-block;
    width: 200px;
    height: 80px;
    background-image: url('imagenes/boton-sprite.svg');
    background-position: 0 0;
    cursor: pointer;
}
.boton-sprite:hover {
    background-position: 0 -80px;
}

Ventajas y desventajas

Ventajas:

  • No requiere JavaScript
  • Mejor rendimiento
  • Las imágenes se precargan automáticamente
  • Transiciones más suaves
  • Con sprites, se reduce el número de solicitudes HTTP

Desventajas:

  • Requiere conocimientos de CSS
  • Los sprites pueden ser más difíciles de mantener
